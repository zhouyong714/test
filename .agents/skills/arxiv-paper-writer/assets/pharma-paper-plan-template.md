## Review Structure (8 Sections)

### 1. Introduction
**Playbook requirements**: Context/Trend → Pain Point → Gap Statement → Scope/Deliverables

- [ ] Why now: Field urgency and opportunity
- [ ] What hurts: Core unsolved problems
- [ ] Gap statement: "However/Yet/Despite..." explicit gap
- [ ] Scope bullets: 3-6 specific deliverables
- [ ] Roadmap paragraph

**Target length**: 1-1.5 pages

### 2. Background & Preliminaries
**Purpose**: Establish foundational concepts for non-specialist readers

- [ ] Drug R&D lifecycle overview (Target → Clinical)
- [ ] AI/ML fundamentals relevant to review scope
- [ ] Key terminology definitions
- [ ] Reader prerequisites

**Target length**: 0.75-1 page

### 3. Taxonomy & Classification
**Playbook requirement**: Must appear in first 10-20% of body

- [ ] Taxonomy figure (decision tree or matrix)
- [ ] Classification dimensions (task × method × stage)
- [ ] Map to subsequent sections
- [ ] Justify pharma-decision-driven organization

**Target length**: 0.5-0.75 pages

### 4-6. Technical Sections (Core Body)
**Playbook requirement**: Each section must follow narrative loop

#### Section 4: `<Task/Method Category 1>`
- [ ] Mechanism (1¶): Why it works
- [ ] Representative Methods (2-4¶): With comparison
- [ ] Failure Modes (≥1¶): When/why it fails
- [ ] Pharma Implications (1¶): Practitioner perspective

#### Section 5: `<Task/Method Category 2>`
- [ ] Mechanism → Methods → Failure → Implications

#### Section 6: `<Task/Method Category 3>`
- [ ] Mechanism → Methods → Failure → Implications

**Target length**: 1.5-2 pages each (4.5-6 total)

### 7. Benchmarks & Evaluation
**Purpose**: Unified benchmark landscape and evaluation practices

- [ ] Benchmark selection rationale (dynamic, not exhaustive)
- [ ] Benchmark comparison table
- [ ] Evaluation metrics and protocols
- [ ] Reproducibility considerations
- [ ] Benchmark limitations and gaps

**Target length**: 1-1.5 pages

### 8. Conclusion & Future Directions
**Playbook requirement**: Actionable roadmap with timeline

- [ ] Key takeaways (3-5 bullets)
- [ ] Short-term directions (6-18 months)
- [ ] Medium-term directions (2-5 years)
- [ ] Reference back to Introduction gap
- [ ] Call to action for community

**Target length**: 0.75-1 page
