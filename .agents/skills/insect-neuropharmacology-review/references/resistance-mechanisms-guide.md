# 抗性机制分类指南

## 四大类抗性机制

### 1. 靶标抗性 (Target-site Resistance)

**定义**: 靶标蛋白结构改变导致杀虫剂亲和力下降

**机制**:
- **点突变**: 单个氨基酸替换
- **基因重复**: 靶标基因拷贝数增加
- **表达量变化**: 靶标蛋白过表达或低表达

**经典案例**:
```yaml
AChE 抗性:
  - G119S: 有机磷/氨基甲酸酯抗性
  - F290V: 有机磷抗性
  - G227A: 氨基甲酸酯抗性

VGSC 抗性 (kdr):
  - L1014F: 拟除虫菊酯抗性
  - M918T: 拟除虫菊酯抗性
  - T929I: 拟除虫菊酯抗性

nAChR 抗性:
  - R81T: 新烟碱类抗性
  - Y151S: 新烟碱类抗性

RDL 抗性:
  - A302S: 氟虫腈抗性
```

### 2. 代谢抗性 (Metabolic Resistance)

**定义**: 解毒酶活性增强,加速杀虫剂降解

**三大酶系**:

**细胞色素 P450 (CYP)**
- 氧化反应
- 最重要的解毒酶系
- 常见基因: CYP6CM1, CYP6BG1, CYP9A61

**谷胱甘肽 S-转移酶 (GST)**
- 结合反应
- 对有机磷、拟除虫菊酯重要
- 常见基因: GSTe2, GSTd1

**羧酸酯酶 (CarE)**
- 水解反应
- 对有机磷、拟除虫菊酯、新烟碱类
- 常见基因: CarE1, CarE2

**机制**:
- 基因过表达
- 基因扩增
- 启动子突变

### 3. 穿透抗性 (Penetration Resistance)

**定义**: 表皮屏障增强,减少杀虫剂吸收

**机制**:
- 表皮增厚
- 脂质组成变化
- 几丁质含量增加

**特点**:
- 非特异性抗性
- 抗性倍数较低 (2-5 倍)
- 常与其他机制协同

### 4. 行为抗性 (Behavioral Resistance)

**定义**: 行为改变避免接触杀虫剂

**类型**:
- 回避行为
- 取食偏好改变
- 栖息地选择

**特点**:
- 难以检测
- 田间重要性高
- 实验室研究少

## 抗性机制组合

### 单一机制
- 抗性倍数: 5-50 倍
- 易被克服

### 多重机制
- 抗性倍数: >100 倍
- 难以克服
- 常见组合:
  - 靶标抗性 + 代谢抗性
  - 多个代谢酶协同

## 抗性检测方法

### 生物测定
- 剂量-反应曲线
- LC50/LD50
- 抗性倍数 (RR)

### 分子检测
- PCR-RFLP
- 实时荧光定量 PCR
- 测序

### 生化检测
- 酶活性测定
- 蛋白表达量

## 抗性管理策略

### 1. 轮换用药
- 不同作用机制
- 避免连续使用

### 2. 混配用药
- 协同增效
- 延缓抗性

### 3. 综合防治
- 生物防治
- 农业防治
- 物理防治

### 4. 抗性监测
- 定期检测
- 早期预警
- 调整策略
